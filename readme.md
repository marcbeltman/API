# Measurment API

> Backend API for Measurment application, wich is a measurment directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

# run in prod mode
npm start
```
